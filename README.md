## 技术栈
- vue2.0+vue-router+vuex（vue全家桶）
- better-scroll(滑动)+vue-lazyload(懒加载)+vue-awesome-swiper(轮播)
- es6（javascript核心规范）
- scss（css预处理器）
- axios（数据请求）
- localStorage （H5本地存储）
